# Healthcare Quality Dataset Analysis

## Project Overview

This repository contains a healthcare quality dataset used for data analysis and machine learning.

The dataset includes:

* Clinician performance scores
* Quality category scores
* Location information
* MIPS performance indicators

## Objectives

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature engineering
* Machine learning modeling

## Dataset Features

* final_MIPS_score
* Quality_category_score
* Cost_category_score
* Improvement_activity_score

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
